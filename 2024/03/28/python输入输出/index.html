python输入输出

输出

print(value,...,sep='',end='\n',file=None) # 如有需要，可以手动修改sep和end

1.在其中可以直接进行计算

2.字符串要打引号（单双引号均可）

3.输出完毕后，自动在末尾添加\n换行

4.一行中输出多个内容使用,作间隔，在表现为一个空格

5.使用chr()函数，可以将acill码转化为字符串进行输出

6.使用open()函数，将内容输出到文件

fp=open('note.txt','w')# 打开文件 w-->write
print('hello world',file=fp)# 将“hello world”输出到note.txt文件中
fp.close()# 关闭文件

7.可以使用+去连接两个字符串（只能是字符串

输入

input('提示文字')

注意：通过input进行输入的数据类型一律默认为字符串类型，若要进行运算之类的，要先进行类型转化

注释

单行注释：#

多行注释：``` 多行注释```

中文文档声明注释：# coding=所需要的编码格式

一定要写在开头第一行

主要是为了防止别人打开你的文件时，由于编码格式不一样，导致中文注释出现乱码
