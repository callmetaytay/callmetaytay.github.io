保留字

查询保留字：

import keyword
print(keyword.kwlist)

保留字严格区分大小写，且不可用作变量名

标识符

命名规则：

1.可以是英文，中文（不建议用中文，下划线，数字，且第一个字符不能是数字

2.严格区分大小写

3.以下划线开头的标识符又特殊意义，一般应避免使用相似的标识符

4.模块名尽量短小，并且全部使用小写字母，可使用下划线分割多个字母

5.包名尽量短小，并且全部使用小写字母，不推荐下划线，可用“.”代替

6.类名采用单词首字母大写的形式（Pascal风格

7.模块内部的类采用“_”和Pascal风格的类名组成

8.函数，类的属性和方法的命名，全部使用小写字母，多个字母之间使用下划线进行分隔

9.常量命名时全部使用大写字母，可使用下划线

10.使用单下划线“_”开头的模块变量或者函数是受保护的，在使用from xxx import xxx 的语句从模块中导入时，这些模块变量和函数不能被导入

11.使用双下划线“__”开头的实例变量或方法是类私有的

12.以双下划线开头和结尾的是Python的专用标识，例如：__init__()表示初始化函数

变量

通过给变量赋不同类型的值，就可以直接修改变量的数据类型

number=8 #此时number的数据类型是int
number="num" #此时number的数据类型是str

允许多个变量同时指向同一个值

number=num=9

数值类型

整数类型

（不可变数据类型）

十进制，二进制，八进制，十六进制

浮点数类型

（不可变数据类型）

表示带有小数点的数值，若对小数位数有需求，使用 round(浮点数，小数位数)

对于复数，虚数的基本单位是j=(-1)^1/2,python中用.real表示整数部分，.imag表示虚数部分

字符串类型

单引号或双引号：单行字符串

三引号：多行字符串

在字符串前面使用”r/R”会使转义字符失效

字符串的索引和切片：

切片：正序是从0开始，逆序是从-1开始

s='helloworld'
print(s[0],s[-10]) #表示的是同一个字符
print('hello'[4]) #获取字符串中索引为4的字符
print(s[2:7]) #从2开始到7结束的字符串（不包括7，正向递增
print(s[-8:-3]) #反向递减
print(s[:5]) #默认从0开始
print(s[5:]) #默认到结尾

常用字符串操作

x='dkfjds'
y='fgpjg'
print(x+y) #连接两个字符串
print(x*4) #将一个字符串重复4遍
print(x in y) #判断x是否为y的子串，返回True或False

布尔类型

用来表示”真“”假“的数据类型，真–>True/1，假–>False/0
