Rethinking Graph Backdoor Attacks: A Distribution-Preserving Perspective

Problem Definition

In-Distribution Constraint on Triggers

Our in-distribution constraint on trigger $g_t$ is defined as :
$$
f_o(g_i) < \tau$$
$f_o(g_i)$ is the anomaly score of $g_i$ and $\tau$ is a threshold which can be tuned based on datasets.

Distribution Preserving Graph Backdoor Attack

This objective si achieved by solving:
$$\min_{\theta_{g}}\sum_{v_i\in\nu_L}l(f_{\theta^*s}(\tilde{v}i),y_t)$$
$$\theta^*s=arg \min{\theta_s}\sum{v_i\in\nu_L}l(f_s(v_i),y_i)+\sum{v_i\in\nu_P}l(f_s(\tilde{v}_i),y_t)$$

Methodology

Chllenges

  1. how to generate ID triggers that have the capability to bypass outlier detection defense methods
  2. how to learn the trigger generator to obtain triggers that meet ID constraint while maintaining a high attack success rate.

In-Distribution Triggers Generation

key: the adversarial learning

Using an MLP as the adaptive trigger generator $f_g$, which generate triggers based on the target node’s node attributes.
$$h^m_i = MLP(x_i), \ \ X^g_i = W_fh^m_i, \ \ A^g_i = W_ah^m_i$$
$x_i$ is the node attributes of $v_i$, and $W_f$, $W_a$ are the learnable parameters for feature and structure generation.

An OOD $f_d$ refines its ability to discern between clean inputs and generated triggers by minimizing the binary classification loss. Concurrently, the generator $f_g$ craft triggers to deceive $f_d$ by maximizing the binary classification loss:
$$\min_{\theta_g}\max_{\theta_d} \mathcal{L}D=\sum{v_i\sim\mathcal{V}s}log(f_d(v_i))+\sum{g\in\tau_P}log(1-f_d(g))$$
To obtain $\mathcal{V}_s$, we pretrain an auto-encoder on the original graph G and select samples whose reconstruction losses are close to the mean loss

Enhancing ID Trigger Effectiveness

Enhancing Memorization of Triggers

Our objective is to ensure that these triggers can guide the surrogate classifier $f_i$ to learn a high cosine similarity between $z_s(\tilde{v}_i)$ and $v_s(\tilde{v}_j)$.
This approach ensures that the trigger attributes significantly impact the target node attributes.

Our goal is to generate triggers that guide the surrogate model to learn a high cosine similarity between enbeddings $z_s(\tilde{v}_i)$ and $z_s(v_j)$, while ensuring the similarity between $z_s(\tilde{v}_i)$ and $z_s(v_k)$ is lower. The loss function is:
$$\mathcal{L}E=-\sum{v_i\in\mathcal{V}U}\sum{v_j\in\mathcal{V}_U}S(z_s(\tilde{v}_i),z_s(\tilde{v}j))+\sum{v_i\in\mathcal{V}U}\sum{v_j\in\mathcal{V}_t}-log\left(\frac{S(z_s(\tilde{v}_i),z_s(v_j))}{S(z_s(\tilde{v}i),z_s(v_j))+\sum{v_k\in\mathcal{V}_L\backslash\mathcal{V}_t}S(z_s(\tilde{v}_i),z_s(v_k))}\right)$$
