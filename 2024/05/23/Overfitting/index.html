Overfitting

产生原因

model的弹性过大，同时training data不足

解决方案

增加training data

1.直接收集数据，增加training data

2.data augmentation：根据数据的特性，人为的创造新的资料。例：若要增加图片类型的数据数量，可以将图片进行左右翻转

constrained model

1.Less parameters and sharing paramenters

2.Less features

3.Early stopping

4.Regularization

5.Dropout

检查

先将training data中取一部分出来作为validation data，剩余的仍作为training data来训练model。训练出来的model用validation data来检验mse是否够小，如果合适了，再放到test data上进行测试
